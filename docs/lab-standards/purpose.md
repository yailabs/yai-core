# Purpose

`docs/lab-standards/` defines a neutral documentation spine for lab work.

The purpose is to make lab writing consistent enough that people and agents can:

- select the correct document type before writing;
- use the required structure for that type;
- distinguish claims, evidence, results, interpretation and limitations;
- link artifacts without inventing data;
- document reproducibility requirements;
- review and freeze documents without changing their meaning.

This directory defines how to document. It does not define what a lab must
study, what a result means, or whether a claim is true.
