# Legal Posture

YAI is source-available for public technical evaluation and review. Public
source access means that technical readers can inspect, evaluate, and test the
repository. It does not mean the project is open source by default.

Unless a file or component explicitly says otherwise, first-party YAI
material is not published under an OSI-approved open-source license. Commercial
use, redistribution, hosted operation, sublicensing, incorporation into
third-party products, and production use require explicit written permission
from the rights holder.

Third-party components remain governed by their own license terms. The current
third-party notice inventory is in `THIRD_PARTY_NOTICES.md`.

Primary legal and repository posture documents:

```text
LICENSE.md
NOTICE.md
THIRD_PARTY_NOTICES.md
CONTRIBUTING.md
SECURITY.md
```

This page is a plain-language summary for repository readers. It is not legal
advice, does not replace the license notice, and still requires legal review
before commercial use, production use, or public launch.
