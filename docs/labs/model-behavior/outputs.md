# Model Behavior Outputs

Status: canonical output contract for curated model behavior artifacts.

## Output Roots

```text
docs/labs/model-behavior/*.md       curated behavior artifacts
docs/labs/model-behavior/raw/       placeholder only; raw local logs stay ignored
docs/labs/model-behavior/reports/   larger curated reports when needed
```

## Curated Artifact Requirements

Each artifact should include:

- status and scope;
- model/provider posture;
- case and subject refs when safe;
- explicit non-authority statement;
- sanitized evidence summary;
- raw capture location policy.

## Non-Goals

Curated behavior outputs are not benchmark reports, security guarantees or
conformance fixtures.
