# Problem

AI-enabled systems are beginning to act on real systems through tools, shell commands, workflow automations, model providers, APIs, repositories and infrastructure controllers.

The gap is not only intelligence. The gap is operational control:

```text
what subject was targeted
what operation was attempted
which policy applied
whether the effect was allowed, denied, observed or imported
which receipt proves what happened
how the result is reconstructed later
what memory is safe to carry forward
```

Existing agent frameworks tend to own planning, tool routing or context assembly. They usually do not provide a local, case-bound control core that can sit beside multiple stacks and produce durable operational evidence.

YAI Core addresses this gap by making attempts, decisions, effects, receipts, graph edges, memory and projections first-class machine objects.
