# Protocols

This is the compact active protocol index. Historical protocol sketches are
archived under `work/archive/protocols/`.

Current protocol families:

- operation attempt
- control decision
- carrier
- receipt
- divergence
- record
- subject binding
- policy materialization
- projection
- ingest
- memory

The active rule is that protocol shapes must preserve case binding, subject
binding, decision posture, carrier boundary and receipt evidence. Durable
protocol history remains archived; new protocol work should update this index
only when the concept is stable enough for curated documentation.
