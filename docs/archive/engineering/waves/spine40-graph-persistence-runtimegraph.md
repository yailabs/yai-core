# SPINE.40 Graph Persistence / RuntimeGraph Doctrine + Schema

Status: archived delivery record.

as distinct surfaces.

## Doctrine

```text
Persistent truth on disk.
Computational shape in memory.
Vector proximity as candidate discovery.
Compiled projection as model-facing view.
```

Graph persistence stores durable typed relations. RuntimeGraph computes over
the active case. HNSW later finds candidate runtime nodes. Context Compiler
later renders controlled views. Projection does not disappear.
