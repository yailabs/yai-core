# NET Terminology

- Node: a local, localhost, LAN, remote or external capability endpoint.
- Endpoint: addressable transport target for a node.
- Stream: ordered transport events for an invocation or observation.
- Capability: advertised work a node can perform.
- Route: selected transport path and node choice, subject to YAI authority.
- Health: node liveness, readiness or degraded status.
- Invocation: request lifecycle carried through NET.
- Trace: correlation vocabulary for transport inspection.
- Receipt: evidence transported by NET and authorized by YAI.

YAI controls authority.
NET moves streams.
External nodes execute capabilities.
