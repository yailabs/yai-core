# Examples

`examples/` is the root location for small runnable or illustrative examples
when present.

No production example is currently provided unless explicitly added later.

`labs/` remains the place for experiments, runs, generated reports and benchmark
artifacts.

`docs/` remains the public documentation surface.

`work/` remains the internal spine, wave, inventory and status surface.

`examples/` must not contain generated lab artifacts.

`examples/` must not contain fake provider, model or runtime claims.
